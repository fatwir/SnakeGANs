# SnakeGANs
Official repository for the implementation of snakeGAN. This repository contains the demo and code for this paper : https://ieeexplore.ieee.org/document/10095903 <br>
Please find the working demo here:
<p><span style="color:green">The green dots are the generated points</span>, <span style="color:blue">blue points are the target</span> and the <span style="color:red">red stars are the snake points</span>. The generated points move to the target points with the help of snake points.</p>
